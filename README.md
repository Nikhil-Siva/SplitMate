# SplitMate 💸

A modern and responsive **group expense splitter** built to simplify trip and shared expense management. SplitMate helps users track expenses, split costs fairly, manage settlements, and organize multiple trips — all in one clean interface.

Perfect for trips, roommates, college projects, outings, and team expenses.

---

## ✨ Features

* 🧾 Create and manage multiple trips
* 👥 Add and manage members
* 💰 Add expenses with category tracking
* ⚖️ Automatic fair expense splitting
* 📊 Real-time balance calculations
* 🤝 Settlement management with pay tracking
* 🌍 Multi-currency support
* 📈 Expense breakdown dashboard
* 📂 CSV expense export
* 🌙 Dark / Light theme toggle
* 📱 Fully responsive UI
* 🔔 Interactive toast notifications and modals

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**

## 📸 Modules

### Dashboard

* Trip overview
* Expense statistics
* Category-wise expense breakdown

### Members

* Add/remove members
* Balance summary per member

### Expenses

* Add categorized expenses
* Split expenses among selected members
* Filter expenses by category

### Settle Up

* View pending and paid settlements
* Mark transactions as paid
* Per-person financial breakdown

### Trips

* Multi-trip management
* Switch between saved trips

### Currency Converter

* Convert currencies
* Embedded live exchange rate reference

## 🚀 How to Run

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Open the project folder

3. Run `index.html` in your browser

No additional setup or dependencies required.

* Better user experience
